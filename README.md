## 💻 What I Build

### 🥇 Flagship Web Platform
**Arino** — My best & most ambitious project:
- Digital agency website with bilingual support (Arabic/English)
- Services: Web Development, Mobile Apps, UI/UX Design
- Built with React, TypeScript, Vite, Tailwind CSS
- [Live Demo](https://arino-v2.vercel.app) | [Repo](https://github.com/its3li/Arino)

### 📱 Mobile Applications
**Aniro** — Complete Islamic companion app with:
- Prayer times with GPS-based calculations
- Full Quran with Tajweed, Tafsir & Audio recitation
- Azkar, Digital Tasbih & Qibla compass
- **100% offline capability**
- Built with **Next.js + Capacitor** for native Android
- [Live Demo](https://aniro-v3.vercel.app) | [Repo](https://github.com/its3li/Aniro)

### 🤖 AI-Powered Platform
**Salaf AI** — Islamic knowledge search engine:
- Intelligent search through authentic sources
- Image analysis capabilities
- Modern Arabic/English interface
- [Live Demo](https://salaf-ai.vercel.app) | [Repo](https://github.com/its3li/Salaf-AI)

### ⚙️ Developer Tools
**Token-Manger-Bot** — Automation tool for token management:
- Simplifies token handling workflows
- Built with JavaScript for efficiency
- [Repo](https://github.com/its3li/Token-Manger-Bot)
